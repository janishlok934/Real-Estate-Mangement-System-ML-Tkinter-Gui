# 🏡 Smart Real Estate Price Predictor

## 📖 Introduction
This project is a desktop-based application designed to simplify real estate management. It combines a **graphical interface (Tkinter)** with a **machine learning model** to estimate property prices based on user-provided details.

The goal of this project is to demonstrate the integration of **Python GUI development and predictive analytics** in a practical use case.

---

## ✨ Key Highlights
- Interactive desktop application  
- Clean and simple user interface  
- Real-time price prediction  
- Image viewing functionality  
- Structured and modular codebase  

---

## ⚙️ Technologies Used
- **Python**
- **Tkinter** (for GUI)
- **Scikit-learn** (for ML model)
- **Pandas & NumPy** (data handling)
- **Pillow (PIL)** (image processing)

---

## 🧠 Model Information
- **Algorithm:** Linear Regression  
- **Purpose:** Predict property prices  
- **Input Parameters:**  
  - Property size  
  - Location  
  - Number of rooms  
- **Output:** Estimated price
